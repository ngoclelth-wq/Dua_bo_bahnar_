# Đụa Bò Bahnar - Toán Học Vui

Trò chơi giáo dục toán học dựa trên trò chơi dân gian của người Bahnar dành cho học sinh tiểu học.

## 🎮 Cách Chơi
- Trò chơi dành cho hai đội: **Đội A** và **Đội B**.
- Mỗi đội sẽ lần lượt trả lời các câu hỏi toán học.
- Trả lời đúng sẽ giúp chú bò của đội tiến lên phía trước.
- Đội nào đưa bò về đích trước sẽ giành chiến thắng!

## 🛠️ Tính Năng
- **Chế độ Giáo viên**: Cho phép thêm, sửa, xóa câu hỏi trong ngân hàng câu hỏi.
- **Âm thanh sống động**: Nhạc nền và hiệu ứng âm thanh vui nhộn.
- **Giao diện thân thiện**: Thiết kế dựa trên văn hóa dân gian Bahnar, phù hợp với lứa tuổi học sinh.

## 🚀 Cài Đặt và Chạy Dự Án
1.  **Cài đặt dependencies**:
    ```bash
    npm install
    ```
2.  **Chạy ở chế độ phát triển**:
    ```bash
    npm run dev
    ```
3.  **Xây dựng dự án**:
    ```bash
    npm run build
    ```
